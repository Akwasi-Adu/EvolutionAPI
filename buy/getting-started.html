3 Get Your Machine ID

Once the API is running, call this endpoint to retrieve your unique Machine ID:

# Using curl curl http://localhost:5000/api/v1/license/machine-id # Or open in your browser: http://localhost:5000/api/v1/license/machine-id

You'll receive a response like this:

{ "machineId": "XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX", "status": "awaiting_activation" }
Copy your Machine ID - You'll need it for the next step!

5 Activate Your License

You'll receive a license.lic file generated specifically for your Machine ID. Here's how to activate it:

Step 1: Place the license.lic file in the root folder containing your API files (same folder as EvolutionSDKAPIsimp.dll).

Step 2: Either restart the API, or call the reload endpoint to validate and apply the license without restarting:

# Reload license without restarting the API curl -X POST http://localhost:5000/api/v1/license/reload # Or if using Swagger UI, call: POST /api/v1/license/reload

Successful activation response:

{ "status": "activated", "licensedTo": "Your Company Name", "tier": "Professional", "modules": ["GL", "AR", "AP", "Inventory", "Sales", "Purchase", "CRM"], "updatesUntil": "2026-12-17", "maxCompanies": 9, "maxInstallations": 3 }
Important: Keep your license.lic file safe! You'll need it if you reinstall or redeploy the API. The license is bound to your Machine ID.

Can I install on multiple machines?

Developer: 1 installation. Professional: Up to 3 installations. Need more? Contact us about Enterprise licensing.

What if I change servers?

Email us with your new Machine ID and we'll transfer your license. No charge for reasonable transfers.

What Sage Evolution versions are supported?

We support Sage Evolution versions 7.x through the latest release. Check our docs for specific version compatibility.

Do I need to keep my updates subscription?

No. Your perpetual license works forever. The updates subscription ($575/year) is only needed if you want new features or compatibility with future Sage versions.
